1.1 Required packages for these exercises

If you would like to run these exercises again on your own machine, you will need to install the following R packages.

  • Biostrings
  • ShortRead
  • Rsamtools
  • Rsubread
  • mixOmics
  • edgeR
  • limma
  • tidyr
  • ggplot2
